# first-website-template
Ett repo att klona och börja lära sig att sätta upp sin första hemsida med Github Pages. I första lektionerna kommer progressivt det visa dig hur du bygger en enkel hemsida med HTML, CSS och enkel Javascript implementeras. Senare kommer det gå vidare med lektioner som gör den till Webbapp, använder ramverket Svelte och slutligen AI som verktyg.
